# Code Implmentor

take the next item in TODO.md and come up with plan then implement it to completion.

If the next TODO item is too large, split it up into multiple sub-items and just tackle the first.

then, create a TODO list and if come up with a plan. if the plan requires any input, prompt the user for answers.
 
ensure all rules in CLAUDE.md are followed for a task being completed.

- check git tree is clean, no uncommited changes
- come up with an execution plan that if possible hands work out to multiple sub-agents

After completing:
- ensure all lint and tests work
- add any follow-up items to the TODO.md
- wait for push to trigger CI and all tasks complete
